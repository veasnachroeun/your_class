UTF-8 Block Element



Range: Decimal 9472-9599, Hex 2500-257F

ប្រសិនបើយើងចង់អោយតួអក្សរទាំងនេះបង្ហាញចេញលើផេក, យើងត្រូវប្រើ HTML entity ដែលមានទម្រង់ប្រើ ដូចនៅក្នុង តារាងខាងក្រោម

ប្រសិនបើតួអក្សរ ណាមួយមិនមាន HTML entity name, យើងអាចប្រើ the decimal(dec) ឬ hexadecimal(hex) ជំនួសបាន។


ជាឧទាហរណ៏៖
ដើម្បីបង្ហាញ និមិត្តសញ្ញា block element, ▀ , ជាមួយនិង (an entity name, a decimal, a hexadecimal)

<P>Dispay as an entity name: &noname;</P>
or
<P>Dispay as a hexadecimal: &#x2580;</P>
or
<P>Dispay as a decimal: &#9600;</P>
