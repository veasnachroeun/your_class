អ្វីទៅជា HTML?

(HTML)មកពីពាក្រថា Hyper Text Markup Languageដែលជាភាសារ Program មួយ ត្រូវបានយកមកប្រើប្រាស់ យ៉ាងទូលំទូលាយក្នុងការបង្កើតចេញជាវេបសាយ។ HTMLត្រូវបានបង្កើតឡើងដោយ Berners-Lee. កំឡុងឆ្នាំ ១៩៩១ ប៉ុន្តែ "HTML 2.0" ត្រូវបានបញ្ជាក់ជា HTML ស្តង់ដាដំបូងគេបង្អស់ ក្នងការដាក់អោយប្រើប្រាស់ជា លើកដំបូង ក្នុងឆ្នាំ ១៩៩៥។ HTML 4.01 ត្រូវបានជំនូសដំណែងថ្មី បន្ទាប់ពី HTML 2.0 វាត្រូវបានគេយក មកប្រើប្រាស់ក្នុង កំឡុងឆ្នាំ ១៩៩៩។ ទោះបីជាជំនាន់ HTML 4.01 ត្រូវបានប្រើប្រាស់យ៉ាងទូលំទូលាយយ៉ាងណាក៏ដោយ ប៉ុន្តែបច្ចុប្បន្ន យើងមានជំនាន់ថ្មី១ទៀង គឺ HTML 5 ដែលបន្ថែមពីលើ HTML 4.01 ហើយវាត្រូវដាក់អោយប្រើ ប្រាស់ក្នុងឆ្នាំ ២០១២។

ស្វែងយល់ HTML Coding ជំហ៊ានដំបូង

ឧទាហរណ៏

<!DOCTYPE html>
<html>
<head>
<title>Index.html</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My First paragraph</p>

</body>
</html>

ស្គាល់ពីមុខងារមួយចំនួនក្នុង HTML 5

  • The <!DOCTYPE html>ជាការប្រកាសដោយកំណត់ថា ឯកសារទាំងនេះជារបស់ html 5
  • The <html> Element ជាធាតុដើមនៃទំព័រ html
  • The <head> Element ផ្ទុករាល់ពត៌មានរបស់ meta អំពី HTML Page
  • The <title> Element កំណត់បង្ហាញជាចំណង់ជើង នៃ html page (ដែលត្រូវបង្ហាញនៅលើ browser's Page tab)
  • The <body> Element សម្គាល់ឯកសារជាតួវេបសាយ, ហើយផ្ទុករាល់ពត៌មានទាំងអស់ដែលអាចមើលឃើញ មានដូចជា heading,paragraph, Image, Hyperlink, table, list, ។ល។
  • The <h1> Element កំណត់បង្ហាញ ជាចំណងជើងធំ
  • The <p> Element កំណត់បង្ហាញជាអត្ថបទធម្មតា

អ្វីជា html element?

An html element កំណត់សម្គាល់ដោយ ការបិតនិងបើក tag, ផ្នែកខាងក្នុងរវាង tag បើក និង tag បិត ជាបណ្តំអក្សរ ឬ អ្វីផ្សេងៗទៀត ទាំងនេះហៅថា html elemnt។

😀 <tagname>ដាក់បញ្ចូលទីនេះ....</tagname>
(html element ត្រូវបានកំណត់ដោយ ការចាប់ផ្តើមនៃ ការបើក tag និង បញ្ចប់ដោយការបិត tag)

< h1>My First Heading< /h1>

< p>My First Paragraph< /p>
